# Akan name gerator


## Author
Louis

## Description
Akan Name Generator is a web application that helps users discover their traditional Akan name based on the day of the week they were born.
In Akan culture, children are given names according to the weekday they are born. This application allows users to enter their birthdate and gender, then calculates the correct Akan name instantly.


## Set up
1. Download or clone the project
2. Open the folder
3. double click on the index.html to open it in your browser
